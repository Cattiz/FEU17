HTML o CSS - Lektion 1

Introduktion

Genomgång av vad HTML är och viktiga begrepp att känna till. Skillnader mellan HTML, CSS och Javascript.

Hur taggar är uppbyggda och viktiga taggar som vi kommer att använda oss utav.
Här är några exempel på taggar:

  • h1- h6 - används för rubriker
  • a - "anchor" används för att lägga till en länk.
  • img - används för att lägga in bilder på webbsidan.
